# 创建虚拟环境

在每一个项目开启 flask 开发前 创建单独的虚拟环境

```bash
cmd bash

windows:
pip install virtualenv virtualenvwrapper-win

查看虚拟环境
workon

创建新的虚拟环境: mkvirtualenv
mkvirtualenv flask2env

删除虚拟环境: rmvirtualenv

进入(切换)虚拟环境
worko flask2env
```
